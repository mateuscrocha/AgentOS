# Idioma Padrão

## Regra Global

Neste projeto, todo texto deve ser produzido por padrao em **portugues brasileiro**.

Isso vale para:

- respostas do kernel
- briefs
- copy
- roteiros
- CTAs
- prompts
- documentacao do usuario
- notas de edicao
- locucao

So usar outro idioma quando o usuario pedir isso explicitamente.

## Regra de qualidade de texto

Todo texto em portugues brasileiro deve sair com:

- acentuacao correta
- uso correto de til, acento agudo, acento circunflexo e cedilha quando aplicavel
- pontuacao natural de fala e leitura
- fraseado claro, nativo e coerente com o contexto

## Regra de locucao

Quando o texto for usado em audio:

- escrever em portugues brasileiro corretamente acentuado e pontuado
- preservar pausas, respiracao e curva de fala
- evitar leitura flat, apatica ou mecanica
- usar emocao e intencao coerentes com a cena

## Excecao

Se o usuario pedir outro idioma, o pedido explicito prevalece para aquela entrega.
