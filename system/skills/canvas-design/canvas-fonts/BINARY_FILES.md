# Canvas Fonts - Binary Files

This directory contains font files (.ttf) and their corresponding OFL license files (.txt) from the Anthropic skills repository. The .ttf font files are binary and were not downloaded. To use them, download from:

https://github.com/anthropics/skills/tree/main/skills/canvas-design/canvas-fonts
