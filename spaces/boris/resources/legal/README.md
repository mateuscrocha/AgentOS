# Biblioteca Jurídica do Bóris

Este diretório reúne os documentos-base de prontidão jurídico-operacional do Bóris.

## Objetivo

Concentrar a documentação pública e interna mínima para reduzir risco de go-live, com foco em:

- LGPD
- termos de uso
- política de privacidade
- transparência sobre coleta e tratamento
- checklist de publicação e operação

## Arquivos iniciais

- `politica-de-privacidade-boris-v1-2026-07-01.md`
- `termos-de-uso-boris-v1-2026-07-01.md`
- `checklist-publicacao-juridica.md`

## Regras de uso

1. Estes documentos são baseline operacional do Bóris, não substituem revisão jurídica especializada.
2. Todo item marcado como `PENDENTE DE DEFINIÇÃO` deve ser resolvido antes da publicação pública.
3. Quando houver mudança material de produto, coleta, terceiros ou retenção, revisar os documentos no mesmo ciclo.
4. Dados institucionais da empresa devem continuar tendo como fonte primária o space `empresa`.
