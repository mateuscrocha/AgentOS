# Reuniões do Bóris

Este diretório é o acervo oficial de reuniões relevantes para o Bóris dentro do AgentOS.

## O que entra aqui

- transcrições de calls
- resumos pós-reunião
- decisões registradas
- próximos passos
- sinais de dor, objeção, oportunidade ou melhoria

## Regra

Se a conversa puder gerar conteúdo, atividade, melhoria de produto, mudança comercial, fluxo operacional ou aprendizado de suporte, ela deve ser consolidada aqui.

## Convenção sugerida

- usar data ISO no início do nome
- incluir conta, pessoa ou tema principal
- preferir um arquivo ou pasta por reunião

Exemplos:

- `2026-04-06-bilheteria-digital-guilherme-marra-matheus-prado.md`
- `2026-04-07-revisao-piloto-boris-analytics.md`

## Relação com outros blocos

O registro bruto ou consolidado fica aqui, mas os aprendizados também devem ser propagados para o destino certo:

- comercial
- produto
- conteúdo
- suporte
- operações
