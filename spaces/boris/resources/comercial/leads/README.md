# Leads Comerciais

Use este diretorio para a base viva de leads do Boris.

## Coloque aqui

- exports CSV
- snapshots de pipeline
- listas segmentadas
- planilhas de prospeccao
- consolidacoes periodicas da base

## Regra

Arquivos aqui sao base operacional. Interpretacao, prioridade da semana e foco atual devem ser refletidos no `world.md` da area comercial.

## Arquivos ja importados

- `boris-base-leads-template.csv`: template inicial de base de leads trazido do legado da Enkrateia
- `leads-master-resumo-legado.txt`: resumo consolidado da base legado com recorte P1, P3 e follow-up pendente
- `top10-leads-quentes-nao-clientes-legado.csv`: fila legado de quentes nao clientes pronta para abordagem
- `snapshot-legado-base-2026-04-06.md`: leitura oficial da base legado trazida para o AgentOS
- `snapshot-crm-atual-2026-04-06.md`: leitura oficial do CRM atual do painel trazida para o AgentOS
- `fila-execucao-comercial-2026-04-06.md`: primeira fila oficial de execucao combinando CRM atual e quentes do legado
