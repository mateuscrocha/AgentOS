# Editais

Coloque aqui os PDFs recebidos e, ao lado deles, crie um resumo em Markdown com:

- nome oficial do edital
- órgão responsável
- objetivo
- público elegível
- tipo de recurso
- exigências e contrapartidas
- prazo
- leitura de aderência ao Bóris
- decisão sugerida: perseguir, monitorar ou descartar
