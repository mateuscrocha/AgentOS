# Pré-Proposta

Esta pasta reúne os materiais de apresentação da frente ENAP.

## Conteúdo

- `apresentacao-web/`: pré-proposta em HTML, CSS e JS
- `semana-de-inovacao-2026/`: projetos separados para entrada do Bóris na Semana de Inovação 2026

## Uso

Use esta área para:

- ajustar narrativa institucional
- adaptar a proposta para Vanessa, ENAP, Rede InovaGov ou editais
- gerar versões de envio e demonstração
- comparar formatos de entrada do Bóris em eventos e chamadas públicas
