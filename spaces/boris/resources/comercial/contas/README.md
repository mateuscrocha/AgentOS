# Contas Comerciais

Use este diretorio para notas por conta, oportunidade ou empresa relevante.

## Formato sugerido

Para casos simples, crie um arquivo por conta relevante, por exemplo:

- `nome-da-empresa.md`

Para oportunidades mais complexas, institucionais ou documentais, pode ser melhor criar uma pasta propria com `README.md` e subpastas de apoio, por exemplo:

- `nome-da-oportunidade/`

## Estrutura sugerida por arquivo

- contexto da conta
- dor percebida
- estagio atual
- pessoas envolvidas
- reunioes e interacoes
- objecoes e travas
- proximo passo
- risco de perda ou potencial estrategico
