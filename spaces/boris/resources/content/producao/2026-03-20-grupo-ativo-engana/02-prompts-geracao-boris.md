# Prompts de Geracao do Boris

Use estes prompts para gerar takes curtos do Boris como apresentador. Evite pedir interface detalhada dentro do video gerado. A UI deve entrar como captura real no Canva.

## Prompt base do personagem

Um mascote/apresentador chamado Boris, assistente inteligente para grupos de WhatsApp. Visual amigavel, expressivo, confiante, carismatico, estilo startup SaaS, visual limpo, formas claras, aparencia consistente entre takes, enquadramento vertical 9:16, iluminacao suave de estudio, fundo simples e moderno, pronto para video curto de produto, falando para camera como apresentador.

## Take 1 - abertura

Vertical 9:16. Boris aparece no centro, olhando direto para a camera, com expressao de alerta e curiosidade, faz um gesto curto com a mao como quem chama atencao para um problema importante. Fundo limpo e elegante, sem elementos distrativos. Espaco livre na parte superior para titulo e na parte inferior para legendas. Estilo de video curto para Reels e Shorts. Duracao sugerida: 3 segundos.

## Take 2 - dor

Vertical 9:16. Boris em meio corpo, levemente deslocado para a esquerda, falando de forma direta e seria, com pequenos gestos de explicacao. Fundo simples com atmosfera de volume e movimento, mas sem texto legivel e sem interface detalhada. Espaco do lado direito para sobreposicao de elementos visuais. Duracao sugerida: 6 a 8 segundos.

## Take 3 - fechamento

Vertical 9:16. Boris volta em destaque, olhando para a camera com expressao de conclusao e clareza, faz um gesto final de apontar ou afirmar. Fundo minimalista. Espaco central para frase final grande. Duracao sugerida: 5 a 6 segundos.

## Prompt para imagem fixa do Boris

Retrato vertical 9:16 do Boris, mascote inteligente para produto SaaS de analise de grupos de WhatsApp, estilo consistente, amigavel, confiante, moderno, visual limpo, fundo neutro, composicao pensada para sobrepor texto, alta qualidade, ideal para thumbnail ou frame de abertura.

## Regras de consistencia
- manter o mesmo estilo visual em todos os takes
- usar sempre fundo simples
- nao gerar dashboard falso com texto pequeno
- deixar areas livres para titulo e legenda
- evitar excesso de elementos cenograficos
