# Template de Campanha

Use este modelo para uma sequencia editorial.

## Arquivos-base

- `00-brief-campanha.md`
- `01-mapa-editorial.md`
- `02-assets-gerais-checklist.md`

## Desdobramento por dia

Quando um dia da campanha virar producao real, crie uma subpasta:

- `01-dia-slug`
- `02-dia-slug`
- `03-dia-slug`

Dentro de cada subpasta, reutilize a estrutura do template de `post`.
