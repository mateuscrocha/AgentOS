# Cabeça Quente

Esta pasta guarda a referencia da linha editorial `cabeca quente`.

Use esta pasta para manter a logica da serie, nao para produzir uma peca especifica.

## O que entra aqui

- regras da serie
- estrutura narrativa padrao
- exemplos aprovados
- criterios de persona
- criterios de locucao
- logica de CTA
- observacoes visuais recorrentes

## O que nao entra aqui

- roteiro final de uma peca especifica
- copy final de um post especifico
- prompts de uma entrega do dia
- checklist de assets de uma peca individual

Esses itens devem ficar em `producao/posts/YYYY-MM-DD-slug`.

## Formula da serie

`cabeca quente` parte de uma dor concreta, recente e vivida por uma pessoa real ou persona editorial.

Estrutura padrao:

1. um desabafo ou dor operacional especifica
2. Boris reage a essa dor como resposta direta
3. a peca caminha para clareza e diagnostico
4. fecha com CTA

## Criterios

- a dor deve parecer vivida hoje, ontem ou muito recentemente
- evitar dores genericas ou abstratas demais
- a abertura deve soar como rotina real, nao slogan
- Boris deve responder como Boris, nao como narrador neutro
- a CTA deve fechar a peca com uma acao clara

## Producao

Quando for criar uma nova peca:

1. crie uma pasta propria em `producao/posts/`
2. use slug especifico da dor
3. mantenha o brief, roteiro, copy, locucao e checklist dentro dessa pasta

Exemplo:

- `producao/posts/2026-03-31-cabeca-quente-gestor-no-achismo`
