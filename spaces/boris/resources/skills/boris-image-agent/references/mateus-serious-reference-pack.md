# Mateus Serious Reference Pack

Use this pack whenever Mateus should look serious, analytical, skeptical, intense, or more authoritative without drifting into a generic AI stern face.

## Purpose
These references exist because serious expression accuracy is part of likeness.
When the prompt asks for a serious look, the generation should borrow the real brow tension, mouth neutrality, and eye focus from these photos instead of inventing a generic intense face.

## Recommended Usage
- For serious thumbnails: use one side-angle serious anchor plus one stable identity image from `assets/mateus-reference-pack/`.
- Prefer side-angle serious references over frontal serious references because the editorial system already favors Mateus slightly turned left or right.
- If the result starts looking too severe or synthetic, soften the expression with a more neutral support image instead of forcing a harder face.

## Avoid
- Generic aggressive face that does not match Mateus
- Overly angry eyebrows
- Mouth compression exaggerated beyond the provided references
- Eye expression that looks hostile instead of focused
