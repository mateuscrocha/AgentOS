# Mateus Reference Pack

This pack stabilizes Mateus across angles, expressions, and crop levels.

## Primary Image
- `assets/mateus-reference-primary.png`
- Source alias: `mateus-reference-primary-smile.png`
- Why it is primary: frontal, clear face visibility, natural smile, balanced lighting, and strong overall recognizability

## Supporting Images
- `assets/mateus-reference-pack/mateus-reference-front-neutral.png`
  Use for neutral expression and straight-on presenter framing.
- `assets/mateus-reference-pack/mateus-reference-front-serious.png`
  Use for more analytical, assertive, or high-contrast technical topics.
- `assets/mateus-reference-pack/mateus-reference-tight-front.png`
  Use when face detail and likeness lock matter more than wardrobe.
- `assets/mateus-reference-pack/mateus-reference-left-profile.png`

- `assets/mateus-reference-pack/mateus-reference-right-three-quarter.png`
  Use for three-quarter angle stability and presenter poses.
- `assets/mateus-reference-pack/mateus-reference-right-profile.png`
  Use for right profile and skull/beard contour consistency.

## Recommended Usage
- For Mateus solo thumbnails: use the primary image plus one neutral or serious frontal support image.
- For expressive thumbnails without smile emphasis: use the primary image plus the serious frontal support image.
- For expressive thumbnails with smile emphasis: use the primary image plus one or two references from `references/mateus-smile-reference-pack.md`.
- For expressive thumbnails with serious emphasis: use the primary image plus one or two references from `references/mateus-serious-reference-pack.md`.
- For expressive thumbnails with curiosity, surprise control, or skeptical intrigue: use the primary image plus one or two references from `references/mateus-raised-eyebrow-reference-pack.md`.
- For side-facing compositions: use the primary image plus the relevant profile or three-quarter support image.
- For `Bóris + Mateus` compositions: keep Mateus on the primary image plus one support image that matches the desired head angle.

## Default Hierarchy For Technical Thumbnails
- Mateus first: authority, expertise, recognizability
- Concept second: system, flow, dashboard, handoff, decision, or signal
- Bóris third: brand support, mascot warmth, visual recall
