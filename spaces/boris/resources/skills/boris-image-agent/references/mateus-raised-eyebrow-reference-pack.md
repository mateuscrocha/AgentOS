# Mateus Raised-Eyebrow Reference Pack

Use this pack whenever Mateus should look curious, intrigued, skeptical in a lighter way, surprised in a controlled way, or with one eyebrow raised for thumbnail emphasis.

## Purpose
These references exist because eyebrow asymmetry is part of likeness in expressive thumbnails.
When the prompt asks for a raised eyebrow, the generation should borrow the real brow shape, forehead tension, and eye asymmetry from these photos instead of inventing a cartoonish or distorted expression.

## Best Raised-Eyebrow Anchors
- `assets/mateus-raised-eyebrow-reference-pack/mateus-raised-eyebrow-right-curious.png`
  Strong candidate for the default raised-eyebrow side-angle look. Good for curiosity, doubt, or "sera que funciona?" compositions.
- `assets/mateus-raised-eyebrow-reference-pack/mateus-raised-eyebrow-right-skeptical.png`
  Good for skeptical intrigue, stronger questioning energy, and more pointed thumbnail hooks.

## Additional Reference
- `assets/mateus-raised-eyebrow-reference-pack/mateus-raised-eyebrow-front-direct.png`
  Useful when direct eye contact matters, but should be used carefully because the editorial system usually prefers slight side angles.

## Recommended Usage
- For raised-eyebrow thumbnails: use one side-angle raised-eyebrow anchor plus one stable identity image from `assets/mateus-reference-pack/`.
- Prefer the side-angle raised-eyebrow references over the frontal one because the editorial system already favors Mateus slightly turned left or right.
- Use this pack for controlled surprise or skeptical curiosity, not for comedy exaggeration.

## Avoid
- Cartoonish eyebrow distortion
- Overly arched brow that stops resembling Mateus
- Forehead wrinkles exaggerated beyond the provided references
- Expression drifting from curious/skeptical into silly
