# Mateus Smile Reference Pack

Use this pack whenever Mateus should look genuinely smiling, welcoming, positive, or confident without drifting into a generic AI grin.

## Purpose
These references exist because smile accuracy is part of likeness.
When the prompt asks for a smile, the generation should borrow the real mouth shape, cheek lift, and eye expression from these photos instead of inventing a new smile.

## Recommended Usage
- For smiling thumbnails: use one side-angle smile anchor plus one stable identity image from `assets/mateus-reference-pack/`.
- Prefer `mateus-smile-right-premium-01.png` or `mateus-smile-right-premium-02.png` first when the user wants a smiling thumbnail and likeness quality matters most.
- Prefer side-angle smile references over frontal smile references because the editorial system already favors Mateus slightly turned left or right.
- If the result starts looking too synthetic, dial back from open smile to subtle smile references.

## Avoid
- Generic toothy grin that does not match Mateus
- Overexposed teeth
- Smile exaggerated beyond the provided references
- Eye expression that looks disconnected from the mouth
