# Boris Editorial

Núcleo editorial do Boris dentro do AgentOS.

## Estrutura

- `editorial/strategy/`: source of truth, linha editorial, dores, tom e templates estratégicos
- `editorial/posts/`: registros editoriais e peças-base
- `editorial/calendars/`: calendários e planos editoriais
- `editorial/assets/`: assets editoriais de apoio
- `output/`: saídas visuais já geradas

## Regra

Tudo que define tese, enquadramento, dor, linguagem e estrutura editorial do Boris deve ser encontrado aqui.
